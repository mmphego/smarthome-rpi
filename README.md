HomeAutoPi
# Raspberry Pi Smart Home Automation Project
        
======

## Usage
* sudo python setup.py install
* sudo apt-get install $(grep -vE "^\s*#" Apts/apt-build-requirements.txt | tr "\n" " ")
* sudo pip insrall -r Apts/pip-build-requirements.txt
